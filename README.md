# AI Customer Churn Predictor 📊

**Live Web App:** [Click here to view the live app](https://churn-predictor-dtfpofhtexjp4nypm4tb3u.streamlit.app/)

## Overview
This project is an end-to-end Machine Learning web application designed to predict the likelihood of a customer canceling their subscription (churn). By analyzing customer data such as tenure, monthly charges, and contract type, this tool provides businesses with actionable, real-time risk assessments. 

## Features
* **Live Predictions:** Interactive sliders and dropdowns allow users to input customer data and instantly generate a churn probability.
* **Trained ML Model:** Utilizes a custom machine learning model saved via Joblib.
* **User-Friendly Interface:** Built entirely in Python using the Streamlit framework for a clean, responsive UI.

## Tech Stack
* **Language:** Python
* **Data Manipulation:** Pandas
* **Machine Learning:** Scikit-Learn
* **Web Framework:** Streamlit
* **Deployment:** Streamlit Community Cloud

## How to Run Locally
If you want to run this project on your own machine, follow these steps:

1. Clone this repository.
2. Install the required packages using `pip install -r requirements.txt`.
3. Run the app using the command: `streamlit run app.py`.
